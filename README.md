Acesta este un repozitoriu ce simuleaza un cos de cumparaturi virtual.

Acest repozitoriu contine 4 clase:

1. Clasa Product care defineste ce fel de produs, pretul si cantitatea.
2. Clasa ProductManager care simuleaza o lista de produse, cu metode de adaugarea si stergere a produselor.
3. Clasa Cart care agregeaza clasa ProductManager si simuleaza adaugarea produselor in cos.
4. Main care apeleaza Toate clasele.

 De asemenea acest proiect contine si un fisier gitignore care ignore fisierele config.json, directorul Screen care contine 
 screenshot-urile din proiect si directorul __pycache__
